## 🔴 문제
> [터렛](https://www.acmicpc.net/problem/1002)

<br/>

## 🟡 Sol
```python
import math
T = int(input())

for i in range(T):
    t = list(map(int, input().split()))
    d = math.sqrt((t[3]-t[0])*(t[3]-t[0])+(t[4]-t[1])*(t[4]-t[1]))
    r_sub = abs(t[2]-t[5])
    r_sum = t[2]+t[5]
    if(t[0]==t[3] and t[1]==t[4] and t[2]==t[5]):
        print("-1")
        continue
    if(r_sub<d<r_sum):
        print("2")
        continue
    if(r_sum == d or r_sub == d):
        print("1")
        continue
    if(r_sum<d or d<r_sub or d == 0):
        print("0")
        continue
```
