## 🔴 문제
> [ACM 호텔](https://www.acmicpc.net/problem/10250)

<br/>

## 🟡 Sol
```python
import sys
T = int(input())
arr = []
for i in range(T):
    H, W, N = map(int, sys.stdin.readline().split())
    room = 101
    cnt = 1
    check_1 = 1
    check_2 = 1
    while cnt <= N:
        if(check_1 <= H):
            arr.append(room)
            room += 100
            check_1 += 1
            cnt += 1
        else:
            room = 101 + check_2
            check_2 += 1
            check_1 = 1
    print(arr[-1])
```

## 🟢 풀이
처음 코드짤때는 n//h와 n%h를 사용했지만 반례가 너무많았다.
