## 🔴 문제
> [두 수 비교하기](https://www.acmicpc.net/problem/1330)

<br/>

## 🟡 Sol
```python
a, b = input().split()
a = int(a)
b = int(b)

if(a>b):
    print(">")
elif(a<b):
    print("<")
else:
    print("==")
```
<br/>

## 🟢 풀이
if - elif 문을 사용해 두수 비교
